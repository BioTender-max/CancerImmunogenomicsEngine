# CancerImmunogenomicsEngine

Cancer immunogenomics engine for neoantigen prediction, HLA typing, MHC binding affinity scoring, and immunoediting detection.

## Features

- Somatic mutation-derived neoantigen prediction pipeline
- HLA class I/II typing from sequencing data
- MHC-peptide binding affinity prediction and strong binder classification
- Immunoediting signature detection and clonal depletion analysis
- Immune checkpoint biomarker integration and response prediction

## Results

150 tumors; Median neoantigen=9; Strong MHC binders=4.1%; Immunoedited=23.3%; Neo-TMB r=0.895

## Usage

```bash
pip install numpy scipy matplotlib
python cancer_immunogenomics_engine.py
```
